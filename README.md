# Kie-Hugo-Theme POC

## Description 

Proof of concept for a possible Kie Theme

## Requirements
* node (version 12.x.x or higher) and npm version 6.x.x.  
* yarn package manager (version 1.13.0 or higher).
* Hugo Static Site Generator v0.69.0/extended 

## Development Scripts

## Clone the repo

```bash
$ git clone https://github.com/yuvaraj-1991/kie-hugo-theme.git 
```

## cd in the repo

```bash
$ cd kie-hugo-theme
```

## install dependencies

```bash
$ npm install
```

#### or yarn

```bash
$ yarn install
```

## run application

```bash
$ npm start
```

#### or yarn

```bash
$ yarn start
```

## build application

```bash
$ npm build
```

#### or yarn

```bash
$ yarn build
```
